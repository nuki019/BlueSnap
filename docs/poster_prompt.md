# 蓝心快搭宣传海报构思与完整提示词

## 1. 海报硬性要求

- 尺寸：竖版 70cm x 150cm。
- 文件大小：最终导出 >= 2MB。
- 必须包含：作品名 `蓝心快搭`，宣传语 `把创造的自由还给用户`。
- 建议印刷分辨率：150 DPI 起步；如电脑性能允许，使用 300 DPI。
- 像素建议：
  - 150 DPI：约 4134 x 8858 px。
  - 200 DPI：约 5512 x 11811 px。
  - 300 DPI：约 8268 x 17717 px。
- 建议导出：PNG 或高质量 JPG；用于提交时可同时保留源文件。

## 2. 海报核心创意

主题：校园里的临时需求，一句话变成手机里的轻工具。

视觉中心：
- 一台大尺寸竖屏手机置于画面中上部或中央。
- 手机屏幕内展示蓝心快搭首页的四个校园场景卡：活动筹备、课程协作、求职实习、生活管理。
- 手机周围漂浮 4 类轻工具卡片：报名名单、课程 DDL、求职投递、生活预算。
- 视觉表现要像真实产品海报，不要像科幻概念图。

文字层级：
- 最大文字：蓝心快搭。
- 第二层：把创造的自由还给用户。
- 第三层：一句话，生成你的校园轻工具。
- 底部三条卖点：手机里直接运行 / 功能可确认可修改 / HTML 可导出可分享。

色彩方向：
- 主色：vivo 蓝、清爽白、少量青色。
- 背景：浅色科技感校园空间，干净明亮。
- 避免：深蓝全屏、紫色渐变、过度赛博、杂乱装饰。

构图方向：
- 竖版海报，上方留标题，中部放手机主视觉，下方放卖点和提交信息。
- 不使用流程图。
- 不要出现无意义的箭头链路。
- 所有文字必须清晰、可读、无遮挡。

## 3. 推荐海报文案

作品名：
> 蓝心快搭

宣传语：
> 把创造的自由还给用户

副标题：
> 一句话，生成你的校园轻工具

卖点：
- 想法变应用，手机里直接运行
- 无需开发环境，功能可确认可修改
- WebView 沙箱隔离，HTML 可离线导出和分享

底部小字：
> 校园效率轻工具生成器 | Android + AI + WebView 沙箱
