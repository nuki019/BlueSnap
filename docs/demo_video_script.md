# 蓝心快搭 3 分钟演示视频脚本

> 目标：展示最终 App 形态、UI、交互、核心功能和安全边界。视频控制在 2 分 45 秒到 3 分钟，1080P 竖屏录制，避免流程图，全部使用真实 App 画面。

## 0:00-0:18 开场与定位

画面：首页，底部三入口清晰可见。

旁白：蓝心快搭是一款面向大学生和准职场青年的校园效率轻工具生成器。一句话，把活动筹备、课程协作、求职投递、生活预算这类临时需求，变成手机里可运行、可保存、可分享的轻工具。

操作：轻扫首页，展示输入框、四个校园场景卡、底部 `历史记录 / + / 系统设置`。

## 0:18-0:45 输入需求

画面：首页输入框。

操作：输入 `帮我做一个活动筹备助手，能记录报名、分工、预算和进度`。

旁白：用户不用安装重 App，也不用学低代码。只要描述场景，系统会先理解需求，再给出可控的功能方案。

## 0:45-1:15 方案确认

画面：对话页流式回复，然后进入方案页。

操作：展示功能清单：报名名单、任务分工、预算记录、进度打卡；展示界面风格选项；点击确认生成。

旁白：蓝心快搭不是直接扔出黑盒结果，而是先让用户确认功能清单和界面风格。主流程控制在两步：说需求，确认生成。

## 1:15-1:45 联网生成与等待说明

画面：方案页生成进度。

操作：展示 `理解需求 -> 功能清单与风格 -> 生成 HTML -> 安全检查 -> 加载预览`。

旁白：当前复赛版本使用蓝心大模型生成单文件 HTML 应用，并保留离线 Mock 兜底。生成阶段会显示进度，避免用户只看到空白等待。

## 1:45-2:15 预览与交互

画面：沉浸式 WebView 预览活动筹备助手。

操作：添加一个报名人；添加一个任务；记录一笔预算；切换标签。

旁白：生成结果直接在 App 的 WebView 沙箱里运行，支持本地保存。没有外部 CDN、没有后端托管，也不会让生成页面自己调用外部 API。

## 2:15-2:35 分享与历史

画面：点击右下角悬浮球，展示操作面板，再进入分享成果页和历史记录页。

操作：点击 `分享成果页`，展示生成摘要、导出 HTML、系统分享 HTML；返回后打开历史记录。

旁白：轻工具生成后可以留存在历史记录里，也可以导出或系统分享成 HTML 文件，方便发给同学、组员或评委现场查看。

## 2:35-2:50 第二条稳定演示

画面：首页输入快捷思维导图，生成并预览。

操作：输入 `帮我制作一个快捷思维导图应用`；展示节点编辑、复制大纲、悬浮输入。

旁白：除了活动筹备，蓝心快搭也能覆盖课堂笔记、项目复盘和求职准备。高频场景有固定高质量兜底，确保现场演示稳定。

## 2:50-3:00 设置与安全收束

画面：系统设置页。

操作：展示深色模式、System Prompt、API 状态占位。

旁白：系统设置支持深色模式和生成风格控制。参赛 AppKey 只用于提交 APK 的现场联网演示；DeepSeek 只做本机评测备用。图片、语音和端侧模型能力当前是预留路线，不包装成已实现功能。
