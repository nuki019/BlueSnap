## 2. 复赛叙事

大叙事：蓝心快搭是校园效率轻工具生成器，面向 18-24 岁大学生和准职场青年。  
核心问题：校园里的很多需求不是“做一个大 App”，而是“现在就需要一个刚好够用的小工具”。现有应用太重、太泛，低代码仍有学习门槛，纯代码生成又需要开发环境。  
一句话价值：一句话，把校园里的临时需求变成手机里可运行、可保存、可分享的轻工具。  
黄金样例：校园活动筹备助手，用来证明这套范式真实可用；课程协作、求职实习、生活管理作为扩展场景。

不要在复赛材料中写成已实现：
- 端侧蓝星 3B 本地推理
- 系统 API 桥接调用相机、短信、健康数据
- 一键生成 APK 或应用市场
- 自动恶意代码扫描平台

可以明确展示的差异化：
- 手机端原生入口，不需要电脑和开发环境
- 首页采用校园工具台式入口：蓝色品牌区、2x2 场景卡、底部创造/历史浮层导航
- 需求对话 -> 方案确认 -> HTML 生成 -> WebView 运行的闭环
- 功能清单可开关，用户在生成前能控制范围
- 历史记录可再次打开、删除、分享生成过的轻工具
- WebView 沙箱运行，默认阻断外部跳转和外部资源
- Mock 兜底保障演示稳定，真实模型作为可配置能力
- 结果分发页集中展示应用摘要、导出 HTML 和系统分享，适合轻量工具流转和留存

## 3. 黄金演示链路

演示输入：
> 给迎新活动做一个报名、分工和预算管理工具，要能记录参与同学、负责人、支出金额和筹备进度。

视频脚本（建议 2分30秒）：
1. 0:00-0:15 展示大叙事：创造力不应卡在“不会开发”上；社团活动是具体样例。
2. 0:15-0:35 打开蓝心快搭，点击“活动筹备”或输入上方需求。
3. 0:35-1:05 展示 AI 流式理解和方案确认，强调功能开关。
4. 1:05-1:35 确认生成并进入 WebView 预览。
5. 1:35-2:10 在活动筹备助手中添加报名、任务、预算，展示数据变化。
6. 2:10-2:25 进入“分享成果”页，导出/分享 HTML，说明可离线保存并传给同学。
7. 2:25-2:40 收尾展示差异化：手机端闭环、即开即用、安全沙箱。

真实截图清单：
- 首页：校园场景卡，包含活动筹备、课程协作、求职实习、生活管理
- 对话页：需求被理解并流式回复
- 方案页：报名名单、任务分工、预算记录、进度打卡
- 预览页：活动筹备助手运行中
- 历史页：生成过的轻工具卡片
- 分享页：应用摘要、导出 HTML、系统分享 HTML

第二演示链路：
> 帮我制作一个快捷思维导图应用

用于证明蓝心快搭不只服务活动筹备，还能覆盖课程笔记、项目复盘、求职准备等青年人的长尾工具需求。Mock 模板已固定为节点增删、层级调整、展开收起、本地保存和复制大纲。
